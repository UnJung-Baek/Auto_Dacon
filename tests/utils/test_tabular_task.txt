### Metric Description: Accuracy

**Accuracy** is a classification metric used to measure the proportion of correctly classified instances out of the total instances. It is defined as:

\[
\text{Accuracy} = \frac{\text{Number of Correct Predictions}}{\text{Total Number of Predictions}}
\]

#### Computation Steps:
1. **Count Correct Predictions**: For each instance in the test dataset, compare the predicted label with the actual label. If they match, count it as a correct prediction.
2. **Count Total Predictions**: Determine the total number of instances in the test dataset.
3. **Calculate Accuracy**: Divide the number of correct predictions by the total number of predictions.

#### Example:
- Suppose you have a test dataset with 100 instances.
- Your model correctly predicts the labels for 85 of these instances.
- The accuracy would be calculated as:

\[
\text{Accuracy} = \frac{85}{100} = 0.85 \text{ or } 85\%
\]

This metric provides a straightforward way to evaluate the performance of a classification model, especially when the classes are balanced. However, it may not be the best metric in cases with imbalanced classes, where other metrics like precision, recall, or F1 score might be more appropriate.
</ANS>

```python
def column_types(input_df: pd.DataFrame) -> dict[str, Any]:
    num_cols = input_df.select_dtypes(include=np.number).columns
    datetime_types = ["datetime", "datetime64", "datetime64[ns]", "datetimetz"]
    datetime_cols = input_df.select_dtypes(include=datetime_types).columns
    column_type_dict = defaultdict(str)
    for col in input_df.columns:
        if col in num_cols:
            column_type_dict[col] = "numerical"
        elif col in datetime_cols:
            column_type_dict[col] = "datetime"
        elif input_df[col].dtype == 'object':
            unique_values = input_df[col].nunique()
            if unique_values <= 5:  # Assuming a small number of unique values indicates a categorical variable
                column_type_dict[col] = "categorical"
            else:
                column_type_dict[col] = "text"
    return column_type_dict
```

```python
"""
This script creates a function that takes an input Dataframe and formats it to follow the df_sample_submission format
"""
# useful imports
import os
import numpy as np
import pandas as pd


### Submission format definition
def df_to_submission_format(
    input_df: pd.DataFrame,
) -> pd.DataFrame:
    """
    Format the input_df to the submission format DataFrame
    Args:
        input_df: DataFrame to be formatted
    Return:
        format_submission_df: DataFrame formatted
    """
    # Ensure the input DataFrame has the correct columns
    if 'id' not in input_df.columns or 'label' not in input_df.columns:
        raise ValueError("Input DataFrame must contain 'id' and 'label' columns.")

    # Map the target column to allowed values
    valid_labels = {'high_income', 'low_income'}
    if not set(input_df['label']).issubset(valid_labels):
        raise ValueError(f"Invalid values in 'label' column. Allowed values are {valid_labels}.")

    # Ensure the label column only contains valid values
    input_df['label'] = input_df['label'].map(lambda x: x if x in valid_labels else None)

    # Format the DataFrame for submission
    format_submission_df = input_df[['id', 'label']]

    return format_submission_df
```
