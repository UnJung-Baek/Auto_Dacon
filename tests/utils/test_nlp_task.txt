### Metric Description: Accuracy

**Accuracy** is a common evaluation metric used for classification tasks. It measures the proportion of correctly classified instances out of the total number of instances.

#### Computation:

1. **True Positives (TP)**: The number of instances where the predicted label matches the actual label.
2. **Total Instances (N)**: The total number of instances in the dataset.

The accuracy is calculated using the formula:
\[ \text{Accuracy} = \frac{\text{TP}}{N} \]

#### Steps to Compute Accuracy:

1. **Collect Predictions**: Obtain the predicted labels for each instance in the test set.
2. **Compare with True Labels**: For each instance, compare the predicted label with the true label.
3. **Count Correct Predictions**: Count the number of instances where the predicted label matches the true label.
4. **Calculate Accuracy**: Divide the number of correct predictions by the total number of instances.

#### Example:

- If you have a test set with 100 instances and your model correctly predicts the labels for 85 of them, the accuracy would be:
\[ \text{Accuracy} = \frac{85}{100} = 0.85 \]

This metric provides a straightforward measure of how well the model is performing in terms of making correct predictions.

```python
import os
import numpy as np
```

```python
### Submission format definition
def df_to_submission_format(
    input_df: pd.DataFrame,
) ->  pd.DataFrame:
    """
    Format the input_df to the submission format DataFrame
    Args:
        input_df: DataFrame to be formatted
    Return:
        format_submission_df: DataFrame formatted
    """
    # Ensure the input DataFrame has the required columns
    required_columns = ['id', 'label']
    if not all(col in input_df.columns for col in required_columns):
        raise ValueError(f"Input DataFrame is missing one or more required columns: {required_columns}")

    # Ensure the label column contains only allowed values
    allowed_values = {'positive', 'negative'}
    if not set(input_df['label']).issubset(allowed_values):
        raise ValueError(f"Label column contains invalid values. Allowed values are: {allowed_values}")

    # Create the submission DataFrame
    format_submission_df = input_df[['id', 'label']].copy()

    return format_submission_df
```

```python
"""
This script creates a function that takes the predicted output `y_pred` and the true output `y_true`
and returns the value of the metric corresponding to the task.
"""
# useful imports
import os
import numpy as np
import pandas as pd
from torch import Tensor

# metric definition
def metric_function(
        y_pred: pd.DataFrame | Tensor,
        y_true: pd.DataFrame | Tensor,
) -> float:
    """
    Computes the metric on (a batch of) inputs and returns the result.
    Args:
        y_pred: the predicted target
        y_true: the true target
    """
    # Ensure y_true and y_pred are of the same length
    if len(y_true) != len(y_pred):
        raise ValueError("The length of y_true and y_pred must be the same.")

    # Drop the 'id' column if present
    if 'id' in y_true.columns:
        y_true = y_true.drop(columns=['id'])
    if 'id' in y_pred.columns:
        y_pred = y_pred.drop(columns=['id'])

    # Convert to numpy arrays for easier comparison
    y_true = y_true.values.flatten()
    y_pred = y_pred.values.flatten()

    # Initialize counter for correct predictions
    correct_predictions = 0

    # Iterate and compare each pair of true and predicted labels
    for true_label, pred_label in zip(y_true, y_pred):
        if true_label == pred_label:
            correct_predictions += 1

    # Compute and return the accuracy
    accuracy = correct_predictions / len(y_true)
    return accuracy
```
